## Store 使用
```
  import { localStore, sessionStore } from '@base-stone/librarys' 
  localStore.set("name", {age: 1}) //localStorage
  localStore.get("name")
  sessionStore.set("name", {age: 1}) //sessionStorage
  sessionStore.get("name")
```
